# Dotfiles
## ANTP
* **`ANTP Edit`** is a modified extension adapted for laptop screen sizes.
* **`Tile String.txt`** is file that contains a string with all of my ANTP preferences.

## Adobe Photoshop
* Place **`Required`** folder inside Adobe Photoshop installation directory.
* Select the darkest color theme available in `Edit > Preferences > Interface`.

## Aegis
* Aegis folder contains all the customizations of my phone.

## Aseprite
* Place **`Aseprite`** folder inside `%AppData%`.
* Theme will be available in `Edit > Preferences > Theme`.

## Blender
* Place **`Blender Foundation`** folder inside `%AppData%`.
* Theme will be available in `File > User Preferences > Themes`.

## CMD
* Go to Clink scripts directory (run `clink info` inside cmd to find that file's location).
* Place **`oh-my-posh.lua`** file inside that directory.

## Deemix
* Place **`config.json`** file inside `%AppData%\deemix`.

## ImageGlass
* Place **`Ext-Icons`** folder inside `Program Files\ImageGlass`.
* Place **`Themes`** folder inside `%AppData%\ImageGlass`.

## IntelliJ
* Go to `%AppData%\JetBrains\IntelliJIdea2022.1\plugins\codelytv-theme\lib`.
* Go to `%AppData%\Google\AndroidStudio2021.2\plugins\codelytv-theme\lib`.
* Open `codelytv-theme-2.2.0.jar` with WinRAR and repack it with Codely Purple theme.

## Mp3tag
* Place **`data`** folder inside Mp3tag installation directory.

## Notepad++
* Place **`Notepad++`** folder inside `%AppData%`.
* Theme will be available in `Settings > Style Configurator`.

## Office
* Place **`store`** folder inside `Windows\System32\ssp`.

## PowerShell
* Place **`PowerShell`** folder inside `Documents`.

## RegistryKey
* **`QTTabBarConfig.xml`** is file that contains all of my QTTabBar preferences.
* The other files are for new Windows Installations.

## ShareX
* **`ShareX-13.7.0-backup.sxb`** is file that contains all of my ShareX preferences.

## Steam
* Place **`skins`** folder inside Steam installation directory.

## SweetScape
* Place **`SweetScape`** folder inside `Documents`.

## Taskbar
* Contains my taskbar icons.

## Terminal
* **`settings.json`** is file that contains all of my Terminal preferences.

## Tiles
* **`Tile Template.psd`** is a template for editing tile images.
* The other files are for tile images.

## TranslucentTB
* **`TranslucentTB.json`** is file that contains all of my TranslucentTB preferences.

## Visual Studio
* Double click **`codely_purple.vsix`** to install Codely Purple theme on Visual Studio 2022.
* Place **`purple.codely-theme-color-theme.json`** file inside `%USERPROFILE%\.vscode\extensions\codely.codely-theme-0.0.1\themes`.

## WinRAR
* Place **`rarreg.key`** file inside `Program Files\WinRAR`.

## qBittorrent
* Place **`themes`** folder inside qBittorrent installation directory.
* Theme will be available in `Tools > Options > Behavior`.
