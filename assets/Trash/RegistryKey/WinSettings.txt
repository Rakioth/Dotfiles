Delete 3D Objects Folder

· Go to:
  HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace
· With NameSpace open on the left, right click and delete the following key:
  {0DB7E03F-FC29-4DC6-9020-FF41B59E513A}
· Go to:
  HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace
· With NameSpace open on the left, right click and delete the following key:
  {0DB7E03F-FC29-4DC6-9020-FF41B59E513A}


Delete Network Folder

· Go to:
  HKEY_CLASSES_ROOT\CLSID\{F02C1A0D-BE21-4350-88B0-7367FC96EF3C}\ShellFolder
· Change the owner of the ShellFolder registry key.
· In the right pane of ShellFolder, double click/tap on Attributes to modify it.
· To Remove Network from Navigation Pane Type in b0940064


Remove Network Icon on Lock and Sign-in Screen

· Open the Local Group Policy Editor.
· Go to:
  Computer Configuration\Administrative Templates\System\Logon
· In the right pane of Logon, double click/tap on the Do not display network selection UI policy to edit it.
· Select (dot) Enabled, click/tap on OK


Disable Windows Defender

· Open Windows Settings and type Windows Security settings.
· Go to:
  Virus & threat protection, then under Virus & threat protection settings click on Manage Settings.
· Turn off:
  Real-time protection / Tamper Protection
· Open Task Scheduler.
· Go to:
  Task Scheduler Library\Microsoft\Windows\Windows Defender
· Select all those tasks, right click and Disable them.


Uninstall Microsoft Edge

· Open File Explorer.
· Go to:
  "C:\Program Files (x86)\Microsoft\Edge\Application\"
· Go to the location above and drill down from the number and Install folder.
· Once you find your path go to cmd and type:
  "C:\Program Files (x86)\Microsoft\Edge\Application\92.0.902.62\Installer\setup.exe" --uninstall --system-level --verbose-logging --force-uninstall
· Keep it from coming back…
  Use stopedge.reg file.


How to Disable Windows Updates

· Open the Local Group Policy Editor.
· Go to:
  Computer Configuration\Administrative Templates\Windows Components\Windows Update
· In the right pane of Windows Update, double click/tap on the Configure Automatic Updates policy to edit it.
· Select (dot) Disabled, click/tap on OK


Install NVIDIA Drivers

· Go to:
  https://www.guru3d.com/files-details/display-driver-uninstaller-download.html
· Open DDU and tick all NVIDIA Specific Options.
· Select device type = GPU and NVIDIA.
· Then click on "Clean and restart".
· Go to:
  https://www.techpowerup.com/download/techpowerup-nvcleanstall/
· Open NVCleanstall and tick only Recommended Settings and next Disable Installer Telemetry & Advertising.
